# 📚 쌓인책 꺼내읽기

### ✅ 방식

- 1~2주단위로 책을 정리한다. (팀원들 스케줄에 맞게 유동적으로 조정예정)
- 분량은 주당 책의 1장~2장 정도로 유동적으로 정한다.
- 한 주마다 리딩을 하는 사람 한명이 있고 그 외 멤버들은 책을 읽다가 모르는 점, 기록으로 남겨 두고 싶은 예제 등 자유로운 방식으로 이슈를 추가한다.
- 스터디 시간 전까지 리딩하는 사람은 해당 이슈에 답변을 달고 스터디 시간에 함께 살펴본다.
